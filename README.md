## 📖 Overview

Smart Navigation System is a desktop application developed using **C++** and **Qt Widgets** that combines an intuitive graphical user interface with classical graph algorithms. The application enables users to compute optimal routes, explore graph traversals, generate minimum spanning trees, estimate travel costs, calculate travel time, and save trip information.

The project was built to demonstrate the practical application of:

- Graph Theory
- Data Structures & Algorithms
- Object-Oriented Programming
- Desktop GUI Development
- File Handling

---

# ✨ Features

- 🛣️ Shortest Path using Dijkstra's Algorithm
- 🌐 Breadth-First Search (BFS)
- 🔍 Depth-First Search (DFS)
- 🌳 Minimum Spanning Tree (Prim's Algorithm)
- 📍 Interactive Graph Visualization
- 🚗 Vehicle-wise Travel Time Estimation
- ⛽ Fuel Cost Calculator
- 💾 Trip History Management
- 🎨 Modern Qt Widgets Interface
